<p>Chart 20.20.010 sets forth the dimensional requirements for each <a class="def" href="20.50.032__7d7c0125340167fb3baa660f4bbc7900">land use</a> district except: the <a class="def" href="20.50.016__df45e3fc3b7917353a8933cb8814cd36">Downtown</a> <a class="def" href="20.50.032__7d7c0125340167fb3baa660f4bbc7900">Land Use</a> Districts, the Evergreen Highlands Design District, the Evergreen Highlands Subarea Transportation Improvement Overlay District, the <a class="def" href="20.50.034__f6b7559aeb02bea8b8ad93ead259b901">Medical Institution District</a>, the OLB-OS <a class="def" href="20.50.032__7d7c0125340167fb3baa660f4bbc7900">Land Use</a> District, and the <a class="def" href="20.50.012__5c523377c6da0dd07cd27247e5b9269e">BelRed</a> <a class="def" href="20.50.032__7d7c0125340167fb3baa660f4bbc7900">Land Use</a> Districts. All <a class="def" href="20.50.046__dc4c71563b9bc39a65be853457e6b7b6">structures</a> and activities in the <a class="def" href="20.50.014__57d056ed0984166336b7879c2af3657f">City</a> not located in the above districts shall conform to the dimensional requirements in Chart 20.20.010. Dimensional requirements for the <a class="def" href="20.50.016__df45e3fc3b7917353a8933cb8814cd36">Downtown</a> <a class="def" href="20.50.032__7d7c0125340167fb3baa660f4bbc7900">Land Use</a> Districts are found in LUC <a class="cite" href="20.25A.060" title="20.25A.060&#x2002;Dimensional charts">20.25A.060</a>. Dimensional requirements for the Evergreen Highlands Design District are found in Part <a class="cite" href="20.25F" title="Part 20.25F&#x2002;Evergreen Highlands Design District">20.25F</a> LUC. Dimensional requirements for the Evergreen Highlands Subarea Transportation Improvement Overlay District are found in Part <a class="cite" href="20.25G" title="Part 20.25G&#x2002;Evergreen Highlands Subarea Transportation Improvement Overlay District">20.25G</a> LUC. Dimensional requirements for the <a class="def" href="20.50.034__f6b7559aeb02bea8b8ad93ead259b901">Medical Institution District</a> are found in Part <a class="cite" href="20.25J" title="Part 20.25J&#x2002;Medical Institution District">20.25J</a> LUC. Dimensional requirements for the OLB-OS <a class="def" href="20.50.032__7d7c0125340167fb3baa660f4bbc7900">Land Use</a> District are found in LUC <a class="cite" href="20.25L.030" title="20.25L.030&#x2002;Dimensional requirements">20.25L.030</a>. Dimensional requirements for the <a class="def" href="20.50.012__5c523377c6da0dd07cd27247e5b9269e">BelRed</a> <a class="def" href="20.50.032__7d7c0125340167fb3baa660f4bbc7900">Land Use</a> Districts are found in LUC <a class="cite" href="20.25D.080" title="20.25D.080&#x2002;Dimensional Requirements">20.25D.080</a>. Dimensional requirements for the Eastgate Transit Oriented <a class="def" href="20.50.016__330f49df8243756a8a4dc7f7f7ee6dfe">Development</a> <a class="def" href="20.50.032__7d7c0125340167fb3baa660f4bbc7900">Land Use</a> District are found in LUC <a class="cite" href="20.25P.060.A" title="20.25P.060.A&#x2002;General">20.25P.060.A</a>. Additional special dimensional requirements for designated areas of the <a class="def" href="20.50.014__57d056ed0984166336b7879c2af3657f">City</a> are contained in other parts of the Code as follows:</p>

<section class="subsec subsec3 chunking-none type-Subsec3" id="20.20.070.A.2.a" data-cite="LUC 20.20.070.A.2.a" data-short-cite="20.20.070.A.2.a"><p><span class="header num-only"><span class="num">a.&#x2002;</span></span>The <a class="def" href="20.50.032__eeec6c7a9d2b475c23650b202208b892">lot</a> does not lie within a <a class="def" href="20.50.044__b9542006bb8a4c3e311ae786c403f368">Residential Land Use District</a>; or</p>

<section class="subsec subsec1 chunking-none type-Subsec1" id="20.20.070.B" data-cite="LUC 20.20.070.B" data-short-cite="20.20.070.B"><p class="heading"><span class="header num-only"><span class="num">B.&#x2002;</span></span><a class="def" href="20.50.016__330f49df8243756a8a4dc7f7f7ee6dfe">Development</a> of an individual <a class="def" href="20.50.032__eeec6c7a9d2b475c23650b202208b892">lot</a> failing to meet 70 percent of the area, width or depth requirements of the <a class="def" href="20.50.032__7d7c0125340167fb3baa660f4bbc7900">land use</a> district in which it is located is restricted to a maximum <a class="def" href="20.50.012__aaaf869064e776a0c7007818de03fe71">building height</a> computed by the following formula (see Example A at the end of this section). The maximum <a class="def" href="20.50.012__aaaf869064e776a0c7007818de03fe71">building height</a> resulting from the following formula may be modified up to the maximum height allowed in the underlying <a class="def" href="20.50.032__7d7c0125340167fb3baa660f4bbc7900">land use</a> district through a <a class="def" href="20.50.052__700bc90a2a50660ba168a0ecdc917606">variance</a> pursuant to Part <a class="cite" href="20.30G" title="Part 20.30G&#x2002;Variance from the Land Use Code">20.30G</a> LUC or Part <a class="cite" href="20.30H" title="Part 20.30H&#x2002;(Reserved)">20.30H</a> LUC:</p>



<p><a class="def" href="20.50.012__aaaf869064e776a0c7007818de03fe71">Building Height</a> = 2 x C x H</p>
<p>C = The ratio of potentially <a class="def" href="20.50.012__674088024c70423599f1da909398176b">buildable area</a> (<a class="def" href="20.50.032__eeec6c7a9d2b475c23650b202208b892">lot</a> area less the area of the <a class="def" href="20.50.032__eeec6c7a9d2b475c23650b202208b892">lot</a>&#x2019;s minimum <a class="def" href="20.50.046__454e7351c6ac0768d6108d7cfcee957a">setback</a> requirements) to total <a class="def" href="20.50.032__eeec6c7a9d2b475c23650b202208b892">lot</a> area.</p>
<p>H = The general <a class="def" href="20.50.012__aaaf869064e776a0c7007818de03fe71">building height</a> requirement otherwise applicable to the <a class="def" href="20.50.032__eeec6c7a9d2b475c23650b202208b892">lot</a>.</p>
<p>(Calculation of <a class="def" href="20.50.012__aaaf869064e776a0c7007818de03fe71">building height</a> is in no way intended to suggest a waiver of <a class="def" href="20.50.032__c9151b5f2819a9be2b150813e4befe82">lot coverage</a> requirements imposed elsewhere in the Code. Similarly, calculations are based on basic Code requirements; any <a class="def" href="20.50.052__700bc90a2a50660ba168a0ecdc917606">variances</a> to <a class="def" href="20.50.046__454e7351c6ac0768d6108d7cfcee957a">setbacks</a> of the property do not affect <a class="def" href="20.50.012__aaaf869064e776a0c7007818de03fe71">building height</a> calculation.)</p>
<p>Provided, that in no event shall <a class="def" href="20.50.012__aaaf869064e776a0c7007818de03fe71">building height</a> exceed the <a class="def" href="20.50.012__aaaf869064e776a0c7007818de03fe71">building height</a> requirement otherwise applicable to the <a class="def" href="20.50.032__eeec6c7a9d2b475c23650b202208b892">lot</a>; and provided, that in no event shall a <a class="def" href="20.50.012__aaaf869064e776a0c7007818de03fe71">building height</a> requirement be imposed less than 15 feet.</p>
</section>
<section class="subsec subsec1 chunking-none type-Subsec1" id="20.20.070.C" data-cite="LUC 20.20.070.C" data-short-cite="20.20.070.C"><p><span class="header num-only"><span class="num">C.&#x2002;</span></span>Notwithstanding subsection <a class="cite" href="20.20.070.A" title="20.20.070.A">A</a> of this section, a <a class="def" href="20.50.032__761cd8ff7f5cb328cd6c4a275b1d89a5">nonconforming lot</a> in a <a class="def" href="20.50.044__b9542006bb8a4c3e311ae786c403f368">Residential Land Use District</a> failing to meet or exceed 70 percent of minimum area, width, and depth requirements of the district in which it is located may not be used for a <a class="def" href="20.50.012__8fce6f8b4e5e6db25f6f8c93753b34f8">building site</a> if at any time since the effective date of the ordinance which first established a minimum <a class="def" href="20.50.032__eeec6c7a9d2b475c23650b202208b892">lot</a> area, width, depth, or <a class="def" href="20.50.032__6bdc32536e69b38777ac6c735f11857c">street frontage</a> requirement larger than the <a class="def" href="20.50.032__eeec6c7a9d2b475c23650b202208b892">lot</a> contains or annexation, whichever was later, has a person, partnership, corporation or marital community owning said <a class="def" href="20.50.032__eeec6c7a9d2b475c23650b202208b892">lot</a> simultaneously owning additional contiguous property. Such <a class="def" href="20.50.032__eeec6c7a9d2b475c23650b202208b892">lots</a> must be combined with additional contiguous property sufficient that the area, width and depth of the combined property each meets or exceeds 70 percent of the minimum requirements of the <a class="def" href="20.50.032__7d7c0125340167fb3baa660f4bbc7900">land use</a> district in which the property is located. This subsection does not constitute a waiver of any of the requirements of <a class="def" href="20.50.012__320c973c0b2dad405bde6e3cfd7de485">boundary line adjustment</a> procedure.</p>



</section>
<section class="subsec subsec1 chunking-none type-Subsec1" id="20.20.070.D" data-cite="LUC 20.20.070.D" data-short-cite="20.20.070.D"><p><span class="header num-only"><span class="num">D.&#x2002;</span></span>Any <a class="def" href="20.50.032__761cd8ff7f5cb328cd6c4a275b1d89a5">nonconforming lot</a> used for a <a class="def" href="20.50.012__8fce6f8b4e5e6db25f6f8c93753b34f8">building site</a> must meet the <a class="def" href="20.50.036__0d4b912344e7695d65adf60d53b7bb6b">nonconforming</a> provisions of LUC <a class="cite" href="20.20.560" title="20.20.560&#x2002;Nonconforming uses, structures, and sites&#xA0;&#x2013; General">20.20.560</a>, the <a class="def" href="20.50.012__aaaf869064e776a0c7007818de03fe71">building height</a> requirements of subsection <a class="cite" href="20.20.070.B" title="20.20.070.B">B</a> of this section and the applicable dimensional requirements of LUC <a class="cite" href="20.20.010" title="20.20.010&#x2002;Uses in land use districts dimensional requirements">20.20.010</a> for the district in which it is located, unless a <a class="def" href="20.50.052__700bc90a2a50660ba168a0ecdc917606">variance</a> has been granted pursuant to Part <a class="cite" href="20.30G" title="Part 20.30G&#x2002;Variance from the Land Use Code">20.30G</a> or <a class="cite" href="20.30H" title="Part 20.30H&#x2002;(Reserved)">20.30H</a> LUC or modification has been granted pursuant to LUC <a class="cite" href="20.25H.040.B" title="20.25H.040.B&#x2002;Allowed Modifications to General Dimensional Chart">20.25H.040.B</a>.</p>



</section>
