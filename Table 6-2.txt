<section class="subsec subsec1 chunking-none type-Subsec1" id="19.06.020(B)" data-cite="EMC 19.06.020(B)" data-short-cite="19.06.020(B)"><p><span class="header"><span class="num">B.&#x2002;</span><span class="name">Average Front Setback&#x2014;Residential Structures in Residential Zones.</span> </span>In certain instances, existing residential dwellings do not meet the current front <a class="def" href="19.04.040__7cf5df3b7cffebbad7be2b619ad6964f">setback</a> standard for the applicable <a class="def" href="19.04.020__31c187ca97c063b0f892fe1257387f79">zone</a> district. In such cases, the <a class="def" href="19.04.020__d30e5176357e29420736c4872099fbc5">applicant</a> <a class="def" href="19.04.010">may</a> <a class="def" href="19.04.010">use</a> the average of the existing front <a class="def" href="19.04.030__d84a9cefcef12b0be99c98cdd1295a1a">facade</a> <a class="def" href="19.04.040__7cf5df3b7cffebbad7be2b619ad6964f">setback</a> of the two nearest and adjacent existing residential <a class="def" href="19.04.030__6875b3c9ced67b7881a1af1c98efabd1">buildings</a> on the same side of the <a class="def" href="19.04.020__4464a81a1583b427705846c35c5123bd">street</a> as the minimum required front <a class="def" href="19.04.040__7cf5df3b7cffebbad7be2b619ad6964f">setback</a> for the <a class="def" href="19.04.030__e2888caa81e44b167d7b8b393589ee06">lot</a>. For <a class="def" href="19.04.030__12d1fd17d08427275300697a0abf5be9">corner lots</a>, the <a class="def" href="19.04.020__d30e5176357e29420736c4872099fbc5">applicant</a> <a class="def" href="19.04.010">may</a> <a class="def" href="19.04.010">use</a> the same <a class="def" href="19.04.040__7cf5df3b7cffebbad7be2b619ad6964f">setback</a> as the adjacent <a class="def" href="19.04.030__6875b3c9ced67b7881a1af1c98efabd1">building</a> on the same side of the <a class="def" href="19.04.020__4464a81a1583b427705846c35c5123bd">street</a>. This provision <a class="def" href="19.04.010">shall</a> apply to principal dwellings only. The resulting <a class="def" href="19.04.040__7cf5df3b7cffebbad7be2b619ad6964f">setback</a> <a class="def" href="19.04.010">shall</a> not be less than fifty percent of the required <a class="def" href="19.04.040__7cf5df3b7cffebbad7be2b619ad6964f">setback</a> standard.</p>

<section class="subsec subsec1 chunking-none type-Subsec1" id="19.06.090(B)" data-cite="EMC 19.06.090(B)" data-short-cite="19.06.090(B)"><p><span class="header"><span class="num">B.&#x2002;</span><span class="name">Front and Street (Side) Setback Reduction on Lots With Excess Right-of-Way.</span> </span>An <a class="def" href="19.04.020__d30e5176357e29420736c4872099fbc5">applicant</a> <a class="def" href="19.04.010">may</a> propose, and the <a class="def" href="19.04.020__bd211164397850d5e1613208425d0efd">planning director</a> <a class="def" href="19.04.010">may</a> allow, <a class="def" href="19.04.010">using</a> the <a class="def" href="19.04.020__9629a979496656df0518dd142b01f6e0">review process</a> in EMC Title <a class="cite" href="15" title="Title 15&#x2002;Local Project Review Procedures">15</a>, a reduced front or <a class="def" href="19.04.020__4464a81a1583b427705846c35c5123bd">street</a> (side) <a class="def" href="19.04.040__7cf5df3b7cffebbad7be2b619ad6964f">setback</a> on <a class="def" href="19.04.030__e2888caa81e44b167d7b8b393589ee06">lots</a> determined by the <a class="def" href="19.04.020__a7ab84800f678f74eca124169725e7ee">city engineer</a> to have excess <a class="def" href="19.04.020__96257e78a1310dbb0e62fa2af8398789">right-of-way</a>.</p>




</section>
<section class="subsec subsec1 chunking-none type-Subsec1" id="19.06.090(C)" data-cite="EMC 19.06.090(C)" data-short-cite="19.06.090(C)"><p><span class="header"><span class="num">C.&#x2002;</span><span class="name">Development Standards That Cannot Be Modified.</span> </span>Any standard that is not specifically listed in this section for modification requests, or in EMC <a class="cite" href="19.06.080" title="19.06.080&#x2002;Land divisions&#x2014;Exceptions to minimum lot area, width, depth, frontage and lot coverage">19.06.080</a>, cannot be modified except as permitted in EMC <a class="cite" href="19.41.010" title="19.41.010&#x2002;Variances">19.41.010</a> for variances. <span class="note history history--linked" data-action="click-&gt;levelnodes#openHistory">(Ord. 4102-25 &#xA7;&#xA0;1 (Exh. 1), 2025; Ord. 3774-20 &#xA7;&#xA0;5(F) (Exh. 3), 2020.)</span></p>




</section>




</article>
<article class="level6 chunking-small type-Section has-history" id="19.06.100" data-cite="EMC 19.06.100" data-short-cite="19.06.100" data-history="/EMC/19.06.100/history"><header class=" full-header" data-type="Section" data-search="/EMC/19.06.100/search" data-share="/EMC/19.06.100/share" data-levelnodes-target="header"><h6 class=" inner-header"><span class="num">19.06.100</span> <span class="name">Residential densities.</span></h6></header><div class="track-indicator selected d-none">
	<span class="glyphicons glyphicons-check" aria-hidden="true"></span><span class="text">This section is included in your selections.</span>
</div>




<section class="subsec subsec1 chunking-none type-Subsec1" id="19.06.100(A)" data-cite="EMC 19.06.100(A)" data-short-cite="19.06.100(A)"><p><span class="header"><span class="num">A.&#x2002;</span><span class="name">Minimum Residential Development.</span> </span>Except within historic overlay <a class="def" href="19.04.020__31c187ca97c063b0f892fe1257387f79">zones</a>, residential <a class="def" href="19.04.020__c9b78e17817e2e9155021f0c78fc6ed2">development</a> in the UR7, MU7, MU15, or LI-MU <a class="def" href="19.04.020__31c187ca97c063b0f892fe1257387f79">zones</a> must result in at least three attached <a class="def" href="19.04.050__966fc300b19a16ac907029d697d780b0">dwelling units</a>.</p>




</section>
<section class="subsec subsec1 chunking-none type-Subsec1" id="19.06.100(B)" data-cite="EMC 19.06.100(B)" data-short-cite="19.06.100(B)"><p><span class="header num-only"><span class="num">B.&#x2002;</span></span>Repealed by Ord. 4102-25</p>



</section>
<section class="subsec subsec1 chunking-none type-Subsec1" id="19.06.100(C)" data-cite="EMC 19.06.100(C)" data-short-cite="19.06.100(C)"><p><span class="header num-only"><span class="num">C.&#x2002;</span></span>Repealed by Ord. 4102-25</p>



</section>
<section class="subsec subsec1 chunking-none type-Subsec1" id="19.06.100(D)" data-cite="EMC 19.06.100(D)" data-short-cite="19.06.100(D)"><p><span class="header"><span class="num">D.&#x2002;</span><span class="name">Density Limits in an Historic Overlay Zone.</span> </span>Residential <a class="def" href="19.04.020__c9b78e17817e2e9155021f0c78fc6ed2">development</a> within a UR4 <a class="def" href="19.04.020__31c187ca97c063b0f892fe1257387f79">zone</a> with an historic overlay (see Chapter <a class="cite" href="19.28" title="Ch. 19.28&#x2002;Historic Resources">19.28</a> EMC) <a class="def" href="19.04.010">shall</a> not exceed one <a class="def" href="19.04.050__966fc300b19a16ac907029d697d780b0">dwelling unit</a> per one thousand five hundred square feet of <a class="def" href="19.04.040__3fdc51a2f7059b3e5cd66ebb94977832">lot area</a> (up to twenty-nine <a class="def" href="19.04.050__966fc300b19a16ac907029d697d780b0">dwelling units</a> per acre). <span class="note history history--linked" data-action="click-&gt;levelnodes#openHistory">(Ord. 4102-25 &#xA7;&#xA0;1 (Exh. 1), 2025; Ord. 3774-20 &#xA7;&#xA0;5(F) (Exh. 3), 2020.)</span></p>




</section>




</article>
<article class="level6 chunking-small type-Section has-history" id="19.06.110" data-cite="EMC 19.06.110" data-short-cite="19.06.110" data-history="/EMC/19.06.110/history"><header class=" full-header" data-type="Section" data-search="/EMC/19.06.110/search" data-share="/EMC/19.06.110/share" data-levelnodes-target="header"><h6 class=" inner-header"><span class="num">19.06.110</span> <span class="name">Density and lot size&#x2014;Neighborhood residential-constrained zone.</span></h6></header><div class="track-indicator selected d-none">
	<span class="glyphicons glyphicons-check" aria-hidden="true"></span><span class="text">This section is included in your selections.</span>
</div>




<section class="subsec subsec1 chunking-none type-Subsec1" id="19.06.110(A)" data-cite="EMC 19.06.110(A)" data-short-cite="19.06.110(A)"><p><span class="header"><span class="num">A.&#x2002;</span><span class="name">Overview and Applicability.</span> </span>Attached dwellings <a class="def" href="19.04.010">may</a> be allowed in the NR-C <a class="def" href="19.04.020__31c187ca97c063b0f892fe1257387f79">zone</a>, subject to <a class="def" href="19.04.020__9629a979496656df0518dd142b01f6e0">review processes</a> set forth in Chapter <a class="cite" href="19.05" title="Ch. 19.05&#x2002;Uses">19.05</a> EMC, review criteria in Chapter <a class="cite" href="15.03" title="Ch. 15.03&#x2002;Land Use Decisions, Criteria and Authority">15.03</a> EMC, <a class="def" href="19.04.020__c9b78e17817e2e9155021f0c78fc6ed2">development</a> standards found in Chapter <a class="cite" href="19.08" title="Ch. 19.08&#x2002;Neighborhood Residential Development Standards">19.08</a> EMC, and this section.</p>




</section>
<section class="subsec subsec1 chunking-none type-Subsec1" id="19.06.110(B)" data-cite="EMC 19.06.110(B)" data-short-cite="19.06.110(B)"><p><span class="header num-only"><span class="num">B.&#x2002;</span></span>The maximum <a class="def" href="19.04.040__956f3cfcf437bd5974e62de0bf3136d1">density</a> for dwellings in the NR-C <a class="def" href="19.04.020__31c187ca97c063b0f892fe1257387f79">zone</a> is one <a class="def" href="19.04.050__966fc300b19a16ac907029d697d780b0">dwelling unit</a> per nine thousand square feet of <a class="def" href="19.04.040__3fdc51a2f7059b3e5cd66ebb94977832">lot area</a>, except as provided in subsection <a class="cite" href="19.06.110(C)" title="19.06.110(C)">(C)</a> of this section. No <a class="def" href="19.04.030__e2888caa81e44b167d7b8b393589ee06">lot</a> <a class="def" href="19.04.010">shall</a> have an area less than four thousand square feet except that <a class="def" href="19.04.030__e2888caa81e44b167d7b8b393589ee06">lots</a> with <a class="def" href="19.04.020__f3179e1d20b9236beb6f538f85ea3f8a">alley</a> access <a class="def" href="19.04.010">may</a> have three-thousand-square-foot <a class="def" href="19.04.030__e2888caa81e44b167d7b8b393589ee06">lots</a>.</p>



</section>
<section class="subsec subsec1 chunking-none type-Subsec1" id="19.06.110(C)" data-cite="EMC 19.06.110(C)" data-short-cite="19.06.110(C)"><p><span class="header num-only"><span class="num">C.&#x2002;</span></span>Each <a class="def" href="19.04.030__e2888caa81e44b167d7b8b393589ee06">lot</a> in the NR-C <a class="def" href="19.04.020__31c187ca97c063b0f892fe1257387f79">zone</a> is permitted up to two <a class="def" href="19.04.030__8f2bddf33ee2f448c9ddbefdc2af58d2">accessory dwelling units</a>; such <a class="def" href="19.04.030__8f2bddf33ee2f448c9ddbefdc2af58d2">accessory dwelling units</a> are exempt from the maximum <a class="def" href="19.04.040__956f3cfcf437bd5974e62de0bf3136d1">density</a> in subsection <a class="cite" href="19.06.110(B)" title="19.06.110(B)">(B)</a> of this section. <span class="note history history--linked" data-action="click-&gt;levelnodes#openHistory">(Ord. 4102-25 &#xA7;&#xA0;1 (Exh. 1), 2025; Ord. 3774-20 &#xA7;&#xA0;5(F) (Exh. 3), 2020.)</span></p>



</section>
